# Ecommerce

Simple frontend for an ecommerce site with product browsing, cart management, and user authentication pages.

## Files

- `index.html` - main storefront
- `products.html` - product listing
- `cart.html` - shopping cart
- `login.html` / `register.html` - auth pages
- `admin.html` - admin interface
- `main.js`, `cart.js`, `orders.js`, `auth.js`, `data.js` - page functionality
- `style.css`, `responsive.css` - styling

## Setup

Open the folder in a browser or use a local web server to run the site.
